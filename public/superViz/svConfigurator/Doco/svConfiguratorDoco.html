svConfiguratorParent

The active visualization is able to be configured. This directive provides the UI for that capability. It is the parent directive to the specific directives that actual edit the configuration.

svConfigSrvc

  • VizInConfigType -- string corresponding to the visualization type of the currently selected visualization. Must be a value in MetricMakerSrvc.visualizationTypes
  • VizInConfigModel --the model of the visualization currently selected.

VisualizationType-specific Configurator Directives

  • svBarChartConfig directive
  • svImageConfig directive
  • svLineChartConfig directive
  • svMapConfig directive
  • svPieChartConfig directive
  • svTermCloudConfig directive
  • svTextConfig directive
